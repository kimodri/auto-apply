# 📨 PUP — Automated Internship Application

This is a simple script for anyone studying in PUP and applying for internships.

The university has sent us an excel containing the possible prospects that we can apply to and whose MOA is still valid.

The expected schema of the dataset that you will use here should be:
1. An Excel file
2. Has the following columns:
SIGNATORY	POSITION	CONTACT NUMBER	EMAIL ADDRESS	ADDRESS	VALIDITY	NOTARIZATION DATE	EXPIRY DATE	ACADEMIC PROGRAM	SOURCE (from what college/branches/campuses)	YEAR SUBMITTED TO ARCDO
